# 👋 안녕하세요, 백엔드 개발자 신가을입니다.

6년 동안 웹 서비스의 운영과 신규 기능 개발을 맡으며  
안정적이고 효율적인 시스템을 만드는 일에 집중해왔습니다.  
서비스의 성능 개선, 데이터 구조 설계, 운영 자동화 등  
실무에서 얻은 경험을 바탕으로 꾸준히 성장하고 있습니다.

### 💻 Tech Stack
**Backend**: Java, Spring Boot, JPA, Kafka, Redis, PostgreSQL  
**Infra**: AWS(EC2, S3), Jenkins, Docker, Grafana, Datadog  
**Frontend**: JSP, jQuery, React(기초)  

### 📁 Projects
- **Kafka Flow** – Spring Boot와 Kafka 기반 비동기 메시징 프로젝트  
  이메일과 카카오톡 알림을 안정적으로 전송하도록 설계  
- **Secure Shelter** – 공공데이터 기반 재난 대피소 플랫폼 (2023 해커톤 준우승)  
- **Junction Hackathon** – 실시간 건설현황 데이터 시각화 프로젝트  
- **Meongnyang Wiki** – 반려동물 정보 검색 플랫폼  
  다양한 반려동물 데이터를 통합해 '애완동물 분야의 네이버'를 목표로 개발  
- **Redis Cache Manager** – Redis를 활용한 캐싱 구조 개선 프로젝트  
  자주 호출되는 데이터 응답속도를 최적화해 서버 부하를 줄임
